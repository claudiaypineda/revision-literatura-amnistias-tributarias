# Revisión de literatura sobre amnistías tributarias

Repositorio de trabajo del working paper:

**“Amnistías tributarias, morosidad y desarrollo local: una revisión de la literatura”**

Este repositorio contiene los scripts, bases de datos y archivos complementarios utilizados en el proceso de revisión de literatura y análisis bibliométrico desarrollado a partir de registros obtenidos en Scopus.

## Estructura del repositorio

- `scripts/`: scripts en R utilizados para la consolidación, depuración y análisis de información.
- `datos/`: bases de datos consolidadas utilizadas en el análisis bibliométrico.
- `anexos/`: archivos complementarios y anexos metodológicos.
